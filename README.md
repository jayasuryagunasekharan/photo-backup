# iPhone Media Copier (Windows)

A simple Windows app to copy photos and videos from an iPhone to a PC or external hard drive using a USB cable.

## Requirements
- Windows 10 or 11
- Python 3.9+
- iPhone USB cable
- Unlock iPhone and tap "Trust This Computer"
- Kepp the screen on all the time

## Run (Development)
```bash
cd photo-backup
python app/main.py
